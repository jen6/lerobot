SO-ARM101 소개

TheRobotStudio의 오픈소스 로봇 팔 프로젝트

SO-ARM101은 SO-ARM100의 개선 버전으로, 다음과 같은 특징이 있습니다:

  • 최적화된 배선: 조인트 3에서 발생하던 연결 끊김 문제 해결
  • 다른 기어비: 리더 팔에 최적화된 기어비 모터 사용
  • 새로운 기능: 리더 팔이 팔로워 팔을 실시간으로 따라갈 수 있음

LeRobot 설치

🐍

Python 환경 생성

Python 3.10 이상 필요

bash 
# conda 환경 생성
conda create -y -n lerobot python=3.10
conda activate lerobot
📦

LeRobot 설치

개발 모드로 설치

bash 
# 저장소 클론 및 설치
git clone https://github.com/huggingface/lerobot.git
cd lerobot
pip install -e ".[feetech]"
⚠️
참고: 안정적인 버전을 원한다면 git clone https://github.com/Seeed-Projects/lerobot.git를 사용하세요.
🔌

포트 권한 설정 (Linux)

USB 시리얼 포트 접근 권한

bash 
# dialout 그룹에 사용자 추가
sudo usermod -aG dialout $USER
# 또는 임시로 권한 부여
sudo chmod 666 /dev/ttyACM*

lerobot-setup-motors \

    --robot.port=/dev/ttyACM0

lerobot-setup-motors \
    --teleop.type=so101_leader \

캘리브레이션

🎯

팔로워 팔 캘리브레이션

로봇 팔의 영점 위치 설정

bash 
lerobot-calibrate \
    --robot.type=so101_follower \
    --robot.port=/dev/ttyACM0 \
    --robot.id=my_follower_arm
🕹️

리더 팔 캘리브레이션

텔레오퍼레이터의 영점 위치 설정

bash 
lerobot-calibrate \
    --teleop.type=so101_leader \
    --teleop.port=/dev/ttyACM1 \
    --teleop.id=my_leader_arm
⚠️
문제 해결: "Magnitude exceeds 2047" 오류가 발생하면 전원을 끄고 다시 켠 후 캘리브레이션을 다시 시도하세요. 캘리브레이션 파일은 ~/.cache/huggingface/lerobot/calibration/에 저장됩니다.

🎬

녹화 설정

데이터셋 녹화 파라미터 설정

bash 
lerobot-record \
	    --robot.type=so101_follower \
	    --robot.port=/dev/ttyACM0 \
	    --robot.id=my_follower_arm \
	    --robot.cameras="{ front: {type: opencv, index_or_path: 0, width: 640, height: 480, fps: 30, fourcc: \"MJPG\"}}" \
	    --teleop.type=so101_leader \
	    --teleop.port=/dev/ttyACM1 \
	    --teleop.id=my_leader_arm \
	    --dataset.repo_id=your_username/so101_pick_cube \
	    --dataset.single_task="Pick up the cube and place it in the box" \
	    --dataset.fps=30 \
	    --dataset.num_episodes=50 \
	    --display_data=true

녹화 단축키

녹화 중 사용할 수 있는 키보드 단축키

키 동작
Space 또는 Enter 현재 에피소드 저장하고 다음으로 진행
Backspace 현재 에피소드 재녹화
Escape 녹화 중지 및 저장
⚠️
주의: 실시간 제어 기능은 베타 버전입니다. 안정적인 장시간 녹화는 CLI 모드(`lerobot-record`)를 권장합니다.
💡
권장사항: 50개 이상의 에피소드를 녹화하세요. ACT 학습에는 RTX 3060에서 약 6시간, RTX 4090에서 약 2-3시간이 소요됩니다.

모델 학습

🧠

ACT 모델 학습

Action Chunking Transformer 학습

bash 
lerobot-train \
    --policy.type=act \
    --dataset.repo_id=username/so101_pick_cube \
    --output_dir=outputs/train/act_so101 \
    --steps=100000 \
    --batch_size=8 \
    --wandb.enable=false
🔄

학습 재개

체크포인트에서 학습 이어가기

bash 
lerobot-train \
    --config_path=outputs/train/act_so101/checkpoints/last/pretrained_model/train_config.json \
    --resume=true
🚀

모델 평가 (배포)

학습된 정책으로 로봇 제어

bash 
lerobot-record \
    --robot.type=so101_follower \
    --robot.port=/dev/ttyACM0 \
    --robot.id=my_follower_arm \
    --robot.cameras="{ front: {type: opencv, index_or_path: 0, width: 640, height: 480, fps: 30, fourcc: \"MJPG\"}}" \
    --dataset.repo_id=username/eval_so101 \
    --dataset.single_task="Pick up the cube and place it in the box" \
    --policy.path=outputs/train/act_so101/checkpoints/last/pretrained_model \
    --display_data=false
